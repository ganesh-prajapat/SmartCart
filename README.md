
# SmartCart

SmartCart is a responsive ReactJS dashboard application built using React, Tailwind CSS, and DummyJSON APIs.

The application includes:

* Authentication
* Protected Routes
* User Profile
* Product Listing
* Product Details
* Pagination
* Responsive Dashboard UI

---

## Features
* Login Authentication using DummyJSON Auth API
* Protected Routes
* Responsive Dashboard
* User Profile Page
* Products Listing with Pagination
* Product Detail Page
* Toast Notifications
* Responsive UI using Tailwind CSS
* Context API for Authentication State

---

## Technologies Used

* ReactJS
* Vite
* Tailwind CSS
* React Router DOM
* Axios
* React Toastify

---

## Setup & Installation

* Clone the repository

```bash
git clone https://github.com/ganesh-prajapat/SmartCart.git
```

* Move into project folder

```bash
cd smartcart
```

* Install dependencies

```bash
npm install
```

* Create `.env` file and add

```env
VITE_BASE_URL=https://dummyjson.com
```

* Start development server

```bash
npm run dev
```
